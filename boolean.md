###### 德摩根式

$\overline{xy}$  =  $\bar{x} + \bar{y}$

$\overline{x+y}$  =  $\bar{x}$ · $\bar{y}$

###### 分配律

(x+y)·z = xz + yz

x+yz = (x+y)·(x+z)

###### 吸收律

|                    A(A+B) = A                    |              A+A·B = A               |
| :----------------------------------------------: | :----------------------------------: |
|             **A·($\bar{A}$+B) = AB**             |      **A + $\bar{A}$·B = A+B**       |
| **(A+B)($\bar{A}$+C)(B+C) = (A+B)($\bar{A}$+C)** | **AB+$\bar{A}$C+BC = AB+$\bar{A}$C** |
